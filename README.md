### chat_analyzer

chat_analyzer — Telegram-бот для защиты групповых чатов от спама. Анализирует входящие сообщения с помощью LLM (OpenAI-совместимого API) и автоматически удаляет сообщения, классифицированные как спам.

### Функциональность

- Мониторинг входящих сообщений в групповых чатах.
- Анализ сообщений с использованием LLM (OpenAI-совместимый API через LangChainGo).
- Автоматическое удаление сообщений при обнаружении спама.
- Управление ботом через личные сообщения администратору: `/setprompt`, `/addchat`, `/removechat`, `/listchats`, `/settemperature`, `/status`, `/help`.
- Настройки (системный промпт, список чатов, температура) сохраняются в JSON-файле и применяются без перезапуска.

### Конфигурация

Бот настраивается через переменные окружения (`.env`):

| Переменная        | Описание                                      |
|-------------------|-----------------------------------------------|
| `DEBUG`           | Режим отладки                                 |
| `LLM_URL`         | URL OpenAI-совместимого API                   |
| `LLM_TOKEN`       | API-токен                                     |
| `LLM_MODEL`       | Название модели                               |
| `LLM_TEMPERATURE` | Температура генерации (0.0–1.0)               |
| `TG_TOKEN`        | Токен Telegram-бота                           |
| `TG_CHAT_IDS`     | ID чатов для мониторинга (через запятую)      |
| `TG_ADMIN_USER_ID`| ID пользователя-администратора                |
| `SETTINGS_PATH`   | Путь к файлу настроек (по умолчанию: `settings.json`) |
